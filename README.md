# Programing-School
# 💻 Проект образовательной платформы YtYt

## 📌 О проекте

Данный проект представляет собой современный адаптивный веб-сайт онлайн-школы программирования **YtYt**, предназначенной для людей, которые хотят получить реальные практические навыки в области разработки программного обеспечения, а не просто сертификат.

Сайт разработан с акцентом на удобство использования, современный дизайн и доступность на различных устройствах.

---

## 🚀 Возможности проекта

* 📱 Адаптивный дизайн для мобильных устройств;
* 💻 Полная поддержка компьютеров и ноутбуков;
* 🎨 Современный и минималистичный интерфейс;
* 📚 Информация об изучаемых технологиях;
* 🧑‍💻 Практико-ориентированный подход к обучению;
* ▶️ Блок с демонстрационным видео;
* 🤝 Поддержка и помощь студентам;
* ⚡ Удобная навигация по сайту.

---

## 🛠 Используемые технологии

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Media Queries
* Responsive Design

---

## 📱 Адаптивность

Проект полностью адаптирован для различных устройств:

* 📱 Смартфоны;
* 💻 Ноутбуки;
* 🖥 Персональные компьютеры.

Благодаря использованию технологии **Responsive Design**, интерфейс автоматически подстраивается под размеры экрана пользователя, обеспечивая комфортное взаимодействие с сайтом на любом устройстве.
